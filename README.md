# Silver Frost

Silver Frost is a custom Codex pet: a silver shaded kitten with a soft pixel-style look.

## Files

- `silver-frost/pet.json`
- `silver-frost/spritesheet.webp`

To use it locally, copy the `silver-frost` folder into your Codex pets directory:

```text
%USERPROFILE%\.codex\pets\silver-frost
```
