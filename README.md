# 银霜：Codex 银渐层小猫桌宠

银霜是一只为 Codex 制作的自定义桌面宠物。它是一只圆脸、白毛、浅银毛尖的银渐层小猫，整体是小尺寸像素风，适合放在 Codex 工作区里陪你写代码。

## 一键安装

已安装 Node.js 的用户可以直接运行：

```bash
npx github:xtyooo/codex-pet-silver-frost
```

脚本会自动把银霜安装到你的 Codex 宠物目录：

```text
%USERPROFILE%\.codex\pets\silver-frost
```

安装完成后，重启 Codex，然后在宠物列表里选择 `Silver Frost`。

## 手动安装

如果不想使用 `npx`，也可以手动复制：

1. 下载这个仓库。
2. 把 `silver-frost` 文件夹复制到你的 Codex 宠物目录：

```text
%USERPROFILE%\.codex\pets\silver-frost
```

3. 重启 Codex。
